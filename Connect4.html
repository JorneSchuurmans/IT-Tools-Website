List

Lijsten worden gebruikt om meerdere items in één variabele op te slaan.

Lijsten zijn een van de 4 ingebouwde gegevenstypen in Python die worden gebruikt om verzamelingen gegevens op te slaan, de andere 3 zijn Tuple, Set en Dictionary, allemaal met verschillende kwaliteiten en gebruik.

Lijsten worden gemaakt met vierkante haken:

Voorbeeld:

Creëer een List:

mylist = ["appel", "banaan", "kers"]
print(mylist)

['appel', 'banaan', 'kers']

List Items

Lijst items zijn geordend, kunnen worden gewijzigd en staan dubbele waarden toe.

Lijst items zijn geïndexeerd, het eerste item heeft index [0], het tweede item heeft index [1], het derde item heeft index [2], enz.

Ordered

Als we zeggen dat lijsten zijn geordend, betekent dit dat de items een gedefinieerde volgorde hebben en dat die volgorde niet verandert.

Als je nieuwe items aan een lijst toevoegt, worden de nieuwe items aan het einde van de lijst geplaatst.

Changeable

De lijst kan worden gewijzigd, wat betekent dat we items in een lijst kunnen wijzigen, toevoegen en verwijderen nadat deze is gemaakt.

Allow Duplicates

Aangezien lijsten zijn geïndexeerd, kunnen lijsten items met dezelfde waarde bevatten:

Voorbeeld:

Lijsten staan dubbele waarden toe:

mylist = ["appel", "banaan", "kers", "appel", "kers"]
print(mylist)

['appel', 'banaan', 'kers', 'appel', 'kers']

List Lengte

Gebruik de functie len() om te bepalen hoeveel items een lijst heeft:

Voorbeeld:

Print het aantal items in de list:

mylist = ["appel", "banaan", "kers"]
print(len(mylist))

3

List Items - Data Types

List items kunnen van elk data type zijn:

Voorbeeld:

String, int en boolean data types:

mylist = ["appel", "banaan", "kers"]
mylist = [69, 420, 31]
mylist = [True, True, False]

Een list kan verschillende data types bevatten:

Voorbeeld:

List met strings, integers en boolean values:

mylist = ["appel", 69, False, 420, "kers"]

type()

Vanuit het perspectief van Python worden lijsten gedefinieerd als objecten met het data type 'list':

<class 'list'>

Voorbeeld:

What is de data type van een list?

mylist = ["appel", 69, False, 420, "kers"]
print(type(mylist))

<class 'list'>

De list() constructor

Het is ook mogelijk om de constructor list() te gebruiken bij het aanmaken van een nieuwe list.

Voorbeeld:

Print het aantal items in de list:

mylist = list(("appel", "banaan", "kers")) #let op de dubbele ronde haakjes!!
print(mylist)

['appel', 'banaan', 'kers']

Een functie is een blok code dat alleen wordt uitgevoerd wanneer deze wordt aangeroepen.

Je kan gegevens, ook wel parameters genoemd, in een functie doorgeven.

Een functie kan gegevens als resultaat retourneren.

Een functie creëren

In Python wordt een functie gedefinieerd met het trefwoord def:

Voorbeeld:


def my_function():
	print("Hallo, ik zit in de functie")

Een functie aanroepen

Om een functie aan te roepen, gebruik je de functienaam gevolgd door haakjes:

Voorbeeld:


def my_function():
	print("Hallo, ik zit in de functie.")

my_function()

Hallo, ik zit in de functie

Argumenten

Informatie kan als argumenten worden doorgegeven in functies.

Argumenten worden gespecificeerd tussen haakjes na de functienaam. Je kan zoveel argumenten gescheiden door een komma toevoegen als je wilt.

Het volgende voorbeeld heeft een functie met een argument (voornaam). Wanneer de functie wordt aangeroepen, geven we een voornaam door, die binnen de functie wordt gebruikt om de volledige naam af te drukken:

Voorbeeld:


def my_function(voornaam):
	print(voornaam + " Mostert-Uit de Fles")

my_function("Jan")
my_function("Piet")
my_function("Joris")
Jan Mostert-Uit de Fles
Piet Mostert-Uit de Fles
Joris Mostert-Uit de Fles

Argumenten worden in Python-documentatie vaak afgekort tot args.

Parameters of Argumenten?

De termen parameter en argument kunnen voor hetzelfde gebruikt worden: informatie die aan een functie wordt doorgegeven.


Vanuit het perspectief van een functie:

Een parameter is de variabele die tussen haakjes wordt vermeld in de functiedefinitie.

Een argument is de waarde die naar de functie wordt gestuurd wanneer deze wordt aangeroepen.

Aantal argumenten

Standaard moet een functie worden aangeroepen met het juiste aantal argumenten. Dit betekent dat als je functie 2 argumenten verwacht, je de functie met 2 argumenten moet aanroepen, niet meer en niet minder.

Voorbeeld:

Hier verwacht de functie 2 argumenten en krijgt 2 argumenten:

def my_function(voornaam, familienaam):
	print(voornaam + " " + familienaam)

my_function("Jan", "Mostert-Uit de Fles")

Jan Mostert-Uit de Fles

Als je de functie probeert aan te roepen met 1 of 3 argumenten, krijg je een foutmelding:

Voorbeeld:

Hier verwacht de functie 2 argumenten en krijgt er maar 1:

def my_function(voornaam, familienaam):
	print(voornaam + " " + familienaam)

my_function("Jan")
Traceback (most recent call last):
File "demo.py", line 4, in 	<module	>
	my_function("Jan")
TypeError: my_function() missing 1 required positional argument: 'familienaam'

Arbitrary Argumenten, *args

Als je niet weet hoeveel argumenten aan je functie moet doorgeven, voeg je een * toe voor de parameternaam in de functiedefinitie.

Voorbeeld:

Als het aantal argumenten onbekend is, voeg je een * toe voor de parameternaam:

def my_function(*kids):
	print("De jongste is " + kids[2])

my_function("Jan", "Piet", "Joris")

De jongste is Joris

Arbitrary Argumenten worden in Python-documentatie vaak afgekort tot *args.

Trefwoordargumenten

Je kan ook argumenten verzenden met de syntaxis key = value.

Op deze manier doet de volgorde van de argumenten er niet toe.

Voorbeeld:


def my_function(child3, child2, child1):
	print("De jongste is " + child3)

my_function(child1 = "Jan", child2 = "Piet", child3 = "Joris")

De jongste is Joris

Arbitrary Trefwoordargumenten, **kwargs

Als je niet weet hoeveel trefwoordargumenten je aan je functie moet doorgeven, voeg je twee asterisk toe: ** voor de parameternaam in de functiedefinitie.

Op deze manier ontvangt de functie een woordenboek met argumenten en heeft ze dienovereenkomstig toegang tot de items:

Voorbeeld:

Als het aantal trefwoordargumenten onbekend is, voeg je een dubbele ** toe voor de parameternaam:

def my_function(**kid):
	print("Zijn familienaam	is " + kid["familienaam"])

my_function(voornaam = "Jan", familienaam = "Mostert-Uit de Fles")

Zijn familienaam is Mostert-Uit de Fles

Standaard Parameterwaarde

Het volgende voorbeeld laat zien hoe je een standaard parameterwaarde gebruikt.

Als we de functie zonder argument aanroepen, gebruikt deze de standaardwaarde:

Voorbeeld:


def my_function(land = "Finland"):
	print(Ik kom uit + land)

my_function("Zuid-Korea")
my_function("Noord-Korea")
my_function()
my_function("Zimbabwe")
Ik kom uit Zuid-Korea
Ik kom uit Noord-Korea
Ik kom uit Finland
Ik kom uit Zimbabwe

Een List doorgeven als Argument

Je kan elk gegevenstype argument naar een functie sturen (string, getal, list, dictionary enz.), en het zal binnen de functie als hetzelfde gegevenstype worden behandeld.

Bv. als je een list als argument verzendt, zal het nog steeds een list zijn wanneer het de functie bereikt:

Voorbeeld:


def my_function(food):
	for x in food:
		print(x)

fruits = ["appel", "banaan", "kers"]

my_function(fruits)
appel
banaan
kers

Returnwaarden

Gebruik de return statement om een functie een waarde te laten returnen:

Voorbeeld:


def my_function(x):
	return 5 * x

print(my_function(3))
print(my_function(5))
print(my_function(9))
15
25
45

De Pass statement

Functiedefinities mogen niet leeg zijn, maar als je om een of andere reden een functiedefinitie zonder inhoud hebt, voer dan de pass statement in om te voorkomen dat je een foutmelding krijgt.

Voorbeeld:


def my_function():
	pass

Recursie

Python accepteert ook functie-recursie, wat betekent dat een gedefinieerde functie zichzelf kan aanroepen.

Recursie is een algemeen wiskundig en programmeerconcept. Het betekent dat een functie zichzelf aanroept. Dit heeft als voordeel dat u gegevens kunt doorlopen om een resultaat te bereiken.

De ontwikkelaar moet heel voorzichtig zijn met recursie, aangezien het vrij gemakkelijk kan zijn om een functie te schrijven die nooit stopt, of een functie die overmatige hoeveelheden geheugen of processorkracht gebruikt. Wanneer correct geschreven, kan recursie echter een zeer efficiënte en wiskundig elegante benadering van programmeren zijn.

In dit voorbeeld is tri_recursion() een functie die we hebben gedefinieerd om zichzelf aan te roepen ("recurse"). We gebruiken de k-variabele als de gegevens, die afnemen (-1) elke keer dat we recursief zijn. De recursie eindigt wanneer de voorwaarde niet groter is dan 0 (d.w.z. wanneer deze 0 is).

Voor een nieuwe ontwikkelaar kan het even duren om erachter te komen hoe dit precies werkt, de beste manier om erachter te komen is door het te testen en aan te passen.

Voorbeeld:

Recursie voorbeeld:

def tri_recursion(k):
	if(k > 0):
		result = k + tri_recursion(k - 1)
		print(result)
	else:
		result = 0
	return result

print("Recursie voorbeeld resultaten")
tri_recursion(6)
Recursie voorbeeld resultaten
1
3
6
10
15
21
