Variabelen

Bij het programmeren wordt veel gebruik gemaakt van variabelen. Een variabele is een naam-waarde combinatie die de computer kan gebruiken. De computer slaat een waarde tijdelijk op 'in' een variabele.

Denk aan 'y = 2x', hierin zijn de x en y variabel. Zodra je een waarde aan x geeft kan je er iets mee, bv x = 2 ⇨ y = 2(2) ⇨ y = 4. De x is de naam van de variabele en de waarde is (in dit geval) 2. Bij programmeren doe je iets soortgelijks: eerst definieer (declareer) je de variabele en daarna gebruik je hem.

Voorbeeld:

Creëer een Variabele:

a = 100
print(a)

100

Wat is een herhaling?

Herhaling of iteratie in programmeren betekent dat de uitvoering van bepaalde instructies moet worden herhaald. Daardoor is een lange reeks instructies veel korter en doorgaans gemakkelijker te begrijpen.

Bij het gebruik van herhalingen in programmeren let je er op dat sommige instructies dezelfde of erg vergelijkbaar zijn. Wat is een If-Else Voorwaarde?

Vrijwel elk algoritme wordt beïnvloed door de keuzes van de gebruiker. Soms kan een keuze van een gebruiker ertoe leiden dat bepaalde delen van een programma wel of niet worden doorlopen. Dit wordt ook wel een selectie genoemd. In plaats van een algoritme stap per stap te doorlopen (= sequentie of opeenvolging), worden enkel de stappen doorlopen waarvoor de gebruiker heeft gekozen.

Een selectie bestaat steeds uit een voorwaarde, via de IF-structuur. Deze IF-structuur gaat steeds kijken of er aan een bepaalde voorwaarde wordt voldaan. Bijvoorbeeld: Is de gebruiker oud genoeg? Is het paswoord juist? Moeten er meerdere voorwaarden worden nagekeken?

Als je de ene voorwaarde wilt rechtvaardigen terwijl de andere voorwaarde niet waar is, gebruik je de Python If Else structuur.

Voorbeeld: If-Else Statement

Het algoritme vraagt de leeftijd van de gebruiker. Als deze leeftijd minder is dan 18, krijgt hij een negatieve boodschap. Anders (dus wanneer hij/zij 18 jaar of meer is) krijgt hij de boodschap dat hij mag autorijden indien hij in het bezit is van een geldig rijbewijs.

leeftijd = 18

if (leeftijd < 18):
   message = "U mag geen auto rijden!""

else:
   message = "Indien u een rijbewijs heeft mag u auto rijden!""

print(message)

Indien u een rijbewijs heeft mag u auto rijden!

List Items

Lijst items zijn geordend, kunnen worden gewijzigd en staan dubbele waarden toe.

Lijst items zijn geïndexeerd, het eerste item heeft index [0], het tweede item heeft index [1], het derde item heeft index [2], enz.

Ordered

Als we zeggen dat lijsten zijn geordend, betekent dit dat de items een gedefinieerde volgorde hebben en dat die volgorde niet verandert.

Als je nieuwe items aan een lijst toevoegt, worden de nieuwe items aan het einde van de lijst geplaatst.

Changeable

De lijst kan worden gewijzigd, wat betekent dat we items in een lijst kunnen wijzigen, toevoegen en verwijderen nadat deze is gemaakt.

Allow Duplicates

Aangezien lijsten zijn geïndexeerd, kunnen lijsten items met dezelfde waarde bevatten:

Recursie

Python accepteert ook functie-recursie, wat betekent dat een gedefinieerde functie zichzelf kan aanroepen.

Recursie is een algemeen wiskundig en programmeerconcept. Het betekent dat een functie zichzelf aanroept. Dit heeft als voordeel dat u gegevens kunt doorlopen om een resultaat te bereiken.

De ontwikkelaar moet heel voorzichtig zijn met recursie, aangezien het vrij gemakkelijk kan zijn om een functie te schrijven die nooit stopt, of een functie die overmatige hoeveelheden geheugen of processorkracht gebruikt. Wanneer correct geschreven, kan recursie echter een zeer efficiënte en wiskundig elegante benadering van programmeren zijn.

In dit voorbeeld is tri_recursion() een functie die we hebben gedefinieerd om zichzelf aan te roepen ("recurse"). We gebruiken de k-variabele als de gegevens, die afnemen (-1) elke keer dat we recursief zijn. De recursie eindigt wanneer de voorwaarde niet groter is dan 0 (d.w.z. wanneer deze 0 is).

Voor een nieuwe ontwikkelaar kan het even duren om erachter te komen hoe dit precies werkt, de beste manier om erachter te komen is door het te testen en aan te passen.

Voorbeeld:

Recursie voorbeeld:

def tri_recursion(k):
   if(k > 0):
      result = k + tri_recursion(k - 1)
      print(result)
   else:
      result = 0
   return result

print("Recursie voorbeeld resultaten")
tri_recursion(6)
Recursie voorbeeld resultaten
1
3
6
10
15

Stap 1: Het plaatsen van een bolletje

Om een bolletje te kunnen plaatsen moeten we eerst een functie hebben die het spel toont, toonSpel(veld), en die alle velden leeg maakt, maakLeegVeld().

Het is de bedoeling dat elke speler om de beurt een bolletje in een kolom plaatsen. We maken dus nu de functie plaatsBlokje(kolom, speler) die twee parameters meekrijgt zodat we kunnen bepalen in welke kolom het bolletje moet en welke speler het mag doen.

Stap 2: Checken of er een winnaar is

We hebben nu bolletjes kunnen plaatsen, dat is heel fijn natuurlijk, maar hoe bepalen we wie gewonnen heeft?

De winnaar is degene die voor snelst 4 bolletjes op een rij krijgt (horizontaal, verticaal of diagonaal).

We maken vier functies checkWin(), om de checken of er effectief een winnaar is, checkHorizontaal(), checkVerticaal() en checkDiagonaal() om in elke riching te checken: 

def checkWin():
   return checkHorizontaal() or checkVerticaal() or checkDiagonaal()

def checkHorizontaal():
   for y in range(6):
   teller = 0
   for x in range(6):
      if veld[x][y] != veld[x+1][y] or veld[x][y] == 0: teller = 0
      else: teller+=1
      if teller >= 3: return True
   return False

def checkVerticaal():
   teller = 0
   for x in range(7):
      teller = 0
      for y in range(5):
      if veld[x][y] != veld[x][y+1] or veld[x][y] == 0: teller = 0
      else: teller+=1
      if teller >= 3: return True
   return False

def checkDiagonaal():
   for x in range(4):
      for y in range(3):
      teller = 0
      for i in range(1, 4):
         if veld[x][y] != veld[x+i][y+i] or veld[x][y] == 0: teller = 0
         else: teller+=1
         if teller >= 3: return True
   
   for x in range(4):
      for y in range(3,6):
      teller = 0
      for i in range(1, 4):
         if veld[x][y] != veld[x+i][y-i] or veld[x][y] == 0: teller = 0
         else: teller+=1
         if teller >= 3: return True
   return False
